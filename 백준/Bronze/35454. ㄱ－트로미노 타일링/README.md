### 문제 설명

<p>$2 \times N$ 크기의 직사각형을 ㄱ-트로미노 타일로 빈칸 없이 채우는 방법의 수를 구하는 프로그램을 작성하시오.</p>

### 입력 

 <p>첫 번째 줄에 테스트 케이스의 개수 $T$가 주어진다. $(1 \le T \le 100)$</p>

### 출력
