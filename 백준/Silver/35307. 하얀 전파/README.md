### 문제 설명

<p>세로로 $N$칸, 가로로 $M$칸 크기의 격자가 주어진다. 처음에는 모든 칸이 검은색 칸이다. 당신은 격자에서 $1$개의 칸을 하얀색으로 바꿀 수 있다.</p>

<p>어떤 하얀색 칸의 위치가 $r$행 $c$열인 $(r,c)$에 있다고 하면, $1$초마다 다음 칸이 하얀색 칸으로 변경된다.</p>

<ul>
<li>왼쪽: $(r,c-1)$, $(r,c-2)$, $\cdots$, $(r,c-L)$ 중 검은색 칸</li>
<li>오른쪽: $(r,c+1)$, $(r,c+2)$, $\cdots$, $(r,c+R)$ 중 검은색 칸</li>
<li>위쪽: $(r-1,c)$, $(r-2,c)$, $\cdots$, $(r-U,c)$ 중 검은색 칸</li>
<li>아래쪽: $(r+1,c)$, $(r+2,c)$, $\cdots$, $(r+D,c)$ 중 검은색 칸</li>
</ul>

<p>이 전파는 모든 하얀색 칸에 대해 동시에 진행된다. 또한 전파는 누적되며 격자 밖으로는 확장되지 않는다.</p>

<p>처음에 하얀색으로 바꿀 칸을 적절히 선택했을 때, 다음 두 값을 구해보자.</p>

<ul>
<li>전파가 더 이상 일어나지 않을 때, 격자에서 하얀색 칸의 개수 최댓값 $W_{\max}$</li>
<li>그 최대 개수에 도달하는 데 걸리는 최소 시간 $T_{\min}$초</li>
</ul>

### 입력 

 <p>첫째 줄에 테스트 케이스의 개수 $T$가 주어진다. $(1\le T\le 100\, 000)$</p>

<p>둘째 줄부터 각 테스트 케이스마다 $1$개의 줄에 정수 $N$, $M$, $L$, $R$, $U$, $D$가 공백으로 구분되어 주어진다. $(1\le N,M\le 10^{9}$; $0\le L,R,U,D\le 10^{9})$</p>

### 출력 

 <p>각 테스트 케이스마다 주어진 순서대로 $1$개의 줄에 정수 $W_{\max}$, $T_{\min}$를 공백으로 구분하여 출력한다.</p>
